# agent-skills

Shared repository for agent skills.

## Install

List available skills:

```bash
npx skills add dapi/agent-skills -l
```

Install one skill globally for all agents:

```bash
npx skills add dapi/agent-skills --skill workspace-cli --agent '*' -g -y
```

Install all skills globally for all agents:

```bash
npx skills add dapi/agent-skills --all -g
```
